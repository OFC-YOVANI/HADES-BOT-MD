## 📢 Aviso Importante

> [!IMPORTANT]\
> Este proyecto **no está afiliado, respaldado ni asociado** con `WhatsApp` (Meta Platforms Inc.). Es un desarrollo independiente y no tiene relación oficial con la compañía.

> [!NOTE]\
> **HADES-BOT-MD** está optimizado principalmente para funcionar en **BoxMine World**. Algunas APIs y características especiales solo estarán disponibles si el bot se ejecuta en un entorno **BoxMine World**. Aunque puedes desplegarlo en otros hosting, la experiencia completa y la estabilidad están garantizadas en **BoxMine World**.
